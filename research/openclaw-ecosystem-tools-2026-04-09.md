## Plugin Highlights (from Composio article)

Plugins hook deeper than skills — they modify agent behavior at the framework level:

1. **Cost tracking plugins** — monitor API spend per agent/channel
2. **Secret detection** — scan for leaked credentials in agent output
3. **Git stats** — automatic commit/PR analytics
4. **Docker integration** — container management from chat
5. **API testing** — Postman-like request builders via chat
